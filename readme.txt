Ako nainštalovať projekt:
-vytvoriť čistú inštaláciu laravelu a gitom si natiahnuť z tohto repozitára aktuálnu verziu
-v env si zmeniť údaje na vlastnú db
-spustiť npm install
-spustiť composer install
-spustiť migráciu

Ako spustiť projekt:
-v konzole dať php artisan serve a cez prehliadač otvoriť danú ip
-quiz nie je napojený na žiadny button, treba sa k nemu dostať cez /bonus v url
